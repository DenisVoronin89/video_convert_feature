1. Создаем сеть бэкенд для докера
     docker network create backend

2. Создаем БД:
    Поднимаем контейнер с постгрес:
        ❯docker-compose up -d postgres
    Запускаем контейнер с постгрес:
        ❯docker-compose up -d postgres
    Заходим в консоль контейнера:
        ❯ docker exec -it video_host-postgres-1 bash
    Заходим в консоль постгрес из контейнера:
        sudo -u postgres psql
    Создаем юзера с учетными данными: admin:admin1224
        CREATE USER admin WITH PASSWORD 'admin1224';
        ALTER USER admin WITH SUPERUSER;
    Создать БД
        CREATE DATABASE stt_video_app;

3. Настраиваем AWS S3 (minio)
    Поднимаем МиниО:
        ❯docker-compose up -d minio
    Заходим в браузере
        http://localhost:45127
    Создаем бакет:
        ищем вкладку create bucket и создаем бакет под именем video-service
    Идем во вкладку http://localhost:45127/access-keys и создаем ключи доступа
    Записываем эти ключи, присваиваем новые значения переменным (модули video_handler_worker.py и video_handler_subscriber.py)
        AWS_ACCESS_KEY_ID = "StdyLLebBVvhXA47msMm"
        AWS_SECRET_ACCESS_KEY = "xOGYPd6V8FH7XfzFur4PcPpwLdhynTMWTgz40FH8"

4. Запускаем монолит приложения
     docker-compose up --build

5. Идем по ссылке http://127.0.0.1:8000/docs (НЕ ЗАБЫВАЕМ ПОДПИСКУ И ЛАЙК)))))))))
